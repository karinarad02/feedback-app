# Serverul se gaseste la adresa http://localhost:5000

# Pentru initializare

### `npm init -y`

# Prentru pornire

### `node server.js`

# Creare baza de date

### `npm install express sequelize sqlite3`

### create sequelize.js file

### `npm start`
